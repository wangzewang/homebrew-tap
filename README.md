# homebrew-tap

Homebrew tap for [amoswzw](https://github.com/amoswzw)'s tools.

## Install

```sh
brew tap amoswzw/tap
```

## Casks

### fluxtty

Multi-session developer workspace terminal.

```sh
brew install --cask fluxtty
```

**Homepage:** https://github.com/amoswzw/fluxtty

### stayawake

Menu-bar utility that keeps macOS awake during long-running work.

```sh
brew install --cask stayawake
```

**Homepage:** https://github.com/amoswzw/stayawake
